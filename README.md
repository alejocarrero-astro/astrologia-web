# Análisis Astrológico - Hyleg y Alcocoden

Aplicación web para cálculo astrológico completo basado en la tradición de Ben Ragel.

## Características
- Cálculo de Hyleg y Alcocoden
- Direcciones primarias (1° = 1 año)
- Gráficos de línea de vida
- Exportación de resultados

## Tecnologías
- Python
- Streamlit
- PySwissEph
- Matplotlib
